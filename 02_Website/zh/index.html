Agent Island

信任边界

本地优先，也把锋利边界说清楚。

  • 它是 Mac 本地工具，不是托管服务。
  • 它读取本机会话记录状态，判断任务是在跑、跑完，还是卡住。
  • 自动续跑会使用绕过权限确认的危险参数，只应该挂到你信任的会话上。
  • 触发日志保存在本机 Application Support 目录下。
