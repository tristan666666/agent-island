FAQ

安装前先看清楚。

这里是项目文档的短版：它是什么、在哪里运行、v1.2.1 更新了什么，以及哪些地方需要信任边界。

Agent Island 是什么？

它是给 Claude Code 和 Codex 用户使用的开源 macOS 工具，用来观察本地长任务，并把完成、卡住、触达额度或等待输入这些状态变得可见。

它会跑在云端吗？

不会。它是本机 Mac 应用。会话状态和触发运行日志都留在你的电脑上，除非你主动分享。

v1.2.1 更新了什么？

v1.2.1 版本重点是实时会话状态、前台 Turn alarm、自动续跑控制，以及用于解释注意力状态的 Status guide。

它怎么判断会话状态？

Agent Island 会保守地读取本地 transcript 和用量信号，再映射成 working、your-turn、stalled、rate-limited 或 auth-required 等状态。

哪里需要额外信任？

自动续跑会使用绕过权限确认的参数。只给你已经信任、可重复的本地工作流挂触发器。

更新和安全怎么处理？

版本标签、Sparkle 更新说明、许可证和安全策略都公开在 GitHub，可以直接检查。
