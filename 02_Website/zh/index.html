Local-first, with the sharp edges named.

  • Runs as a Mac utility, not a hosted service.
  • Reads local transcript state to tell whether a session is working, done, or stalled.
  • Auto-resume uses dangerous permission-bypass flags, so only trusted sessions should be attached.
  • Trigger run logs stay under your local Application Support folder.
